SQL Injection
—
악의적 SQL 구문을 입력 값에 삽입해 DB를 비정상적으로 조작하는 공격
ARP Spoofing
Address Resolution Protocol Spoofing
ARP 캐시를 위조하여 트래픽을 가로채는 중간자 공격 기법
DDoS
Distributed Denial of Service
다수의 좀비 PC로 서버에 과부하를 유발하여 서비스를 불가능하게 만드는 공격
SYN Flood
—
대량의 SYN 패킷으로 서버 연결 큐를 가득 채우는 DoS 공격
Zero-Day
—
공개·패치되지 않은 취약점을 이용하는 공격. APT에서 주로 활용
RBAC
Role-Based Access Control
사용자의 역할(Role)에 따라 권한을 부여하는 접근 제어 모델
ABAC
Attribute-Based Access Control
사용자·환경·리소스 속성을 기반으로 접근을 제어하는 정책 모델
Replay Attack
—
이전에 캡처한 유효한 패킷을 재전송하여 불법 인증을 시도하는 공격
MITM
Man-in-the-Middle Attack
통신 당사자 사이에 끼어 트래픽을 도청·변조하는 공격
HIDS
Host-based Intrusion Detection System
호스트 내부 로그·시스템 콜을 모니터링하는 호스트 기반 침입 탐지 시스템
NIDS
Network-based Intrusion Detection System
네트워크 트래픽을 분석하여 침입을 탐지하는 네트워크 기반 IDS
